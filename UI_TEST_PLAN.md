# 實價登錄 SPA - UI 測試計劃

## 1. 測試環境設定
- **前端應用**: Vue 3 + Element Plus SPA (localhost:5173)
- **後端 API**: Node.js dataServer (localhost:3002)
- **測試工具**: Puppeteer (Headless Chrome)

## 3. 風險與注意事項
- **CORS**: 確保 dataServer (3002) 已開啟且 Vue 前端設定正確指向 3002 port。
- **資料完整性**: 確認 `lvr_data.db` 已包含台北與台中數據。
